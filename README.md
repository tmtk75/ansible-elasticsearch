elasticsearch
=============
Ansible role for elasticsearch.
This role ensures that elasticsearch runs as a service.

Requirements
------------
No requirements.

Role Variables
--------------
- consul\_enabled

Dependencies
------------
No dependencies.

Example Playbook
----------------

    - hosts: servers
      roles:
         - { role: tmtk75.elasticsearch }
